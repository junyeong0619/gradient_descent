# 경사하강법 & 몬테카를로 시뮬레이터 구현 보고서

## 1. 프로젝트 개요

비용 함수(Cost Function) 위에서 **모멘텀 경사하강법(Momentum SGD)**이 작동하는 과정을 시각화하고, **몬테카를로(Monte Carlo)** 방법으로 지역 최솟값 문제를 완화하는 원리를 인터랙티브 애니메이션으로 보여주는 시뮬레이터다.

## 2. 핵심 알고리즘

### 2-1. 모멘텀 경사하강법 (Momentum SGD)

일반 경사하강법은 기울기 방향으로만 이동하기 때문에, 비볼록 함수에서는 지역 최솟값에 갇히면 탈출하지 못한다. 모멘텀을 추가하면 이전 이동 방향의 관성이 누적되어 얕은 지역 최솟값을 통과할 수 있다.

**업데이트 규칙:**
```
v_t = β · v_{t-1} + ∇J(w_t)
w_{t+1} = w_t - α · v_t
```

- `α` (LR = 0.005): 학습률. 한 스텝의 이동 크기를 결정한다.
- `β` (MOMENTUM = 0.97): 모멘텀 계수. 1에 가까울수록 관성이 강하다.
- `v`: 속도(velocity). 이전 기울기들의 가중 합으로, 방향성과 가속도 역할을 한다.

**구현 (`simulation.py`):**
```python
def run_gd(w_init):
    w = np.array(w_init, dtype=float)
    v = np.zeros(2)
    for _ in range(N_EPOCHS):
        g = grad_fn(w)          # 수치 기울기 계산
        v = MOMENTUM * v + g    # 속도 누적
        w = w - LR * v          # 가중치 업데이트
```

### 2-2. 수치 기울기 (Numerical Gradient)

데이터로부터 피팅한 비용 함수는 해석적 미분이 불가능하므로, 중앙 차분법(central difference)으로 기울기를 근사한다.

```
∂J/∂w1 ≈ [J(w1+ε, w2) - J(w1-ε, w2)] / 2ε
```

```python
def grad_fn(w, eps=1e-4):
    g1 = (cost_fn(w[0]+eps, w[1]) - cost_fn(w[0]-eps, w[1])) / (2*eps)
    g2 = (cost_fn(w[0], w[1]+eps) - cost_fn(w[0], w[1]-eps)) / (2*eps)
    return np.array([g1, g2])
```

### 2-3. 몬테카를로 방법 (Monte Carlo)

단일 시작점을 쓰는 경사하강법은 초기값에 따라 수렴 결과가 달라진다. 몬테카를로 접근은 탐색 공간 전체에 걸쳐 여러 시작점을 무작위(또는 균등)로 배치하고, 각각 독립적으로 최적화를 실행한다. 이 중 가장 낮은 비용에 도달한 결과를 전역 최솟값 후보로 채택한다.

**구현:**
```python
# 4×4 격자로 16개 시작점 생성
mc_starts = [np.array([w1s, w2s])
             for w1s in np.linspace(W1_RANGE[0]+0.5, W1_RANGE[1]-0.5, 4)
             for w2s in np.linspace(W2_RANGE[0]+0.5, W2_RANGE[1]-0.5, 4)]

mc_hists = [run_gd(s) for s in mc_starts]  # 각 시작점에서 독립 실행
```

## 3. 데이터 연동

### CSV 형식

사용자가 직접 정의한 비용 함수 데이터를 불러올 수 있다. 형식은 아래와 같다.

```
x,y,z
-4.0,-4.0,15.32
-4.0,-3.87,12.10
...
```

- `x`, `y`: 가중치 공간의 좌표 (w1, w2)
- `z`: 해당 좌표에서의 비용값 J(w1, w2)

### 4차 다항식 피팅

샘플 점들로부터 연속적인 비용 함수를 복원하기 위해 **최소자승법(Least Squares)**으로 2변수 4차 다항식을 피팅한다.

피팅에 사용되는 특징(feature) 집합:

```
{w1^i · w2^j  |  i, j ≥ 0,  i + j ≤ 4}
```

총 15개의 단항식으로 구성되며, numpy의 `lstsq`로 계수를 구한다.

```python
def fit_poly_surface(x, y, z, deg=4):
    def features(x, y):
        cols = []
        for i in range(deg + 1):
            for j in range(deg + 1 - i):
                cols.append(x**i * y**j)
        return np.column_stack(cols)

    coeffs, _, _, _ = np.linalg.lstsq(features(x, y), z, rcond=None)
```

데이터가 비볼록 함수에서 샘플링되었다면, 피팅된 다항식도 비볼록 형태를 재현한다.

### 기본 함수 (파일 미선택 시)

Himmelblau 함수의 변형을 기본값으로 사용한다. 4개의 전역 최솟값을 가지는 대칭적 비볼록 함수로, 몬테카를로의 효과를 극적으로 보여주기 적합하다.

```python
J(w1, w2) = [(w1² + w2 - 11)² + (w1 + w2² - 7)²] / 50
```

GLOBAL_THRESH = j_global + (Zg.max() - Zg.min()) * 0.08

## 5. 두 단계 시뮬레이션

### Phase 1 — 단일 시작점 (Momentum SGD)

좌상단 구석에서 출발한 공이 지역 최솟값을 만나더라도, 모멘텀으로 쌓인 관성 덕분에 경사를 올라 탈출하고 전역 최솟값에 도달하는 과정을 보여준다.

**핵심 메시지:** 일반 경사하강법은 지역 최솟값에 갇히지만, 모멘텀을 추가하면 관성으로 탈출할 수 있다.

### Phase 2 — 몬테카를로 (16개 시작점)

전체 탐색 공간에 4×4 격자로 16개 시작점을 배치하고 각각 Momentum SGD를 실행한다. 어떤 시작점은 지역 최솟값에 수렴하고(빨강), 어떤 시작점은 전역 최솟값에 도달한다(초록).

**핵심 메시지:** 시작점을 다양하게 시도함으로써 전역 최솟값을 놓칠 확률을 낮출 수 있다. 이것이 몬테카를로 방법의 실용적 의미다.
